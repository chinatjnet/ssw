ssw
===

ssw站长建站系统 V1.0

专为个人站长打造的傻瓜式建站系统，让你的前台模板不受后台程序的丝毫限制。

功能包括：个人档、个人心情、个人日志、个人相册、网址收藏、个人音乐、资源下载、访客留言、情侣空间、好友管理、访客统计、文件管理……
测试版本V1.0只是完成了整体框架的架构，后续版本会继续完善细化功能。
